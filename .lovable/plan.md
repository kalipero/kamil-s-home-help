

# Kamil Vavřička — Hodinový manžel Portfolio Website

## Overview
A clean, minimalist single-page portfolio website in Czech for a handyman service provider. The site will prioritize readability, trust, and easy contact.

## Design Direction
- **Light background** with dark text for maximum readability
- **Blue accent color** for buttons, icons, and highlights
- **Simple sans-serif typography** with generous spacing
- **Sticky top navigation** for easy section jumping
- **Fully responsive** — mobile-first approach

## Sections

### 1. Navigation Bar (Sticky)
- Logo/name: "Kamil Vavřička"
- Links: Domů, O mně, Služby, Kontakt
- Mobile hamburger menu

### 2. Hero / Home Section
- Large heading: "Kamil Vavřička — Hodinový manžel"
- Short tagline about reliable handyman services
- Placeholder for profile photo
- Two CTA buttons: "Zavolat" (click-to-call) and "Kontaktovat" (scroll to contact)
- Quick icon list of main service categories

### 3. About Section (O mně)
- Brief personal description emphasizing reliability, versatility, and practical help
- What customers can expect when working with Kamil

### 4. Services Section (Služby)
Six service cards in a responsive grid, each with:
- Lucide icon
- Service name and short Czech description
- Cards: Drobné opravy a montáže, Instalace spotřebičů, Údržba a renovace, Instalatérské služby, Elektrikářské práce, Zahradní práce

### 5. Contact Section (Kontakt)
- Phone number with click-to-call button
- Email address
- Location: Hanušovice
- Simple contact form (name, email, message) with validation and success toast

### 6. Footer
- Contact info summary
- Copyright notice

## Technical Notes
- Single-page app with smooth scroll navigation
- No backend needed — contact form will show a success message (can be connected to a real backend later)
- All text in Czech
